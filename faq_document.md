# M7 Portfolio Monitor - FAQ 자주 묻는 질문 ❓

## 📊 전략 관련

### Q1: 왜 3개월 Lookback을 사용하나요?

**A**: Walk-Forward 최적화를 통해 도출된 최적 파라미터입니다.

```
테스트 결과:
- 1개월: 너무 단기, 노이즈 많음
- 3개월: ✅ 최적 (변동성과 수익 균형)
- 6-12개월: 반응 속도 느림
```

### Q2: Monthly 리밸런싱이 Weekly보다 좋은 이유는?

**A**: 거래 비용 대비 성과가 우수합니다.

```
Weekly:
- 연간 회전율: ~400-500%
- 높은 거래 비용
- 과도한 거래

Monthly: ✅
- 연간 회전율: ~150-250%
- 적정 거래 비용
- 안정적 성과
```

### Q3: Threshold -30%는 어떻게 결정되었나요?

**A**: 심각한 하락을 구분하는 기준점입니다.

```
-15%: 경미한 조정 (정상 변동)
-20%: 중간 하락 (주의)
-30%: ✅ 심각한 하락 (기회)
-40%: 극심한 하락 (드묾)
```

### Q4: Weight Split 60%의 의미는?

**A**: 심각한 하락 종목에 집중 투자하는 비율입니다.

```
예시: TSLA -35%, META -25%, 나머지 -5%

Weight Split 60%:
- TSLA, META (Drawdown < -30%): 60% 할당
  → TSLA: 35%, META: 25%
- 나머지 5종목: 40% 할당
```

## 📈 성과 관련

### Q8: 실제 거래 시 수익률이 다른 이유는?

**A**: 거래 비용과 슬리피지가 반영되지 않았기 때문입니다.

```
표시된 수익률: 이론적
실제 수익률 차감 요소:
- 거래 수수료: 약 -0.5%p/year
- 슬리피지: 약 -0.3%p/year
- 세금: 개인별 상이
- 타이밍 차이: 마감가 vs 실제 체결가

예시:
이론 CAGR: 20%
실제 CAGR: ~18.5-19%
```

### Q9: M7 Equal Weight보다 항상 좋은가요?

**A**: 아닙니다. 특정 시기에는 뒤처질 수 있습니다.

```
Dynamic이 좋은 시기:
✅ 변동성 큰 시장
✅ 개별 종목 차별화 심함
✅ 하락-반등 사이클

Equal Weight이 좋은 시기:
⚠️ 강한 상승장 (모든 종목 동반 상승)
⚠️ 낮은 변동성

장기적으로는 Dynamic이 우수
```

### Q10: 과거 성과가 미래를 보장하나요?

**A**: 절대 아닙니다!

```
⚠️ 중요한 면책사항:
- 과거 성과 ≠ 미래 수익
- 시장 환경 변화 가능
- 전략 유효성 감소 가능
- 투자 손실 위험 존재

권장:
- 전략 이해 후 사용
- 분산 투자 병행
- 정기적 모니터링
- 전문가 상담
```

### Q13: CSV 파일을 어떻게 활용하나요?

**A**: Excel, Python 등에서 추가 분석에 사용하세요.

```python
# Python 예시
import pandas as pd

# 포트폴리오 데이터
df = pd.read_csv('portfolio_values_20241111.csv')
df['Date'] = pd.to_datetime(df['Date'])

# 월별 수익률
monthly = df.set_index('Date')['M7_Dynamic'].resample('M').last().pct_change()
print(monthly.describe())

# 승률 계산
win_rate = (monthly > 0).sum() / len(monthly)
print(f"월간 승률: {win_rate:.1%}")
```
